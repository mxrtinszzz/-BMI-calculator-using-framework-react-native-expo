# Calculadora de IMC — App React Native

Este aplicativo foi desenvolvido para calcular o Índice de Massa Corporal (IMC) a partir dos dados informados pelo usuário. Para instalar as dependências do projeto, é necessário abrir o terminal dentro da pasta do projeto e executar o comando npm install. Caso o Expo ainda não esteja instalado no computador, também será preciso instalar globalmente utilizando o comando npm install -g expo-cli.

Para executar o projeto, utilize o comando npx expo start no terminal. Após iniciar, o aplicativo poderá ser aberto no celular através do aplicativo Expo Go, no navegador pressionando a tecla W ou em um emulador Android pressionando a tecla A.

Para usar o aplicativo, basta digitar o peso em quilogramas e a altura em metros nos campos indicados e pressionar o botão Calcular. O sistema exibirá automaticamente o valor do IMC, a classificação correspondente e uma cor indicativa que representa o nível do resultado.
